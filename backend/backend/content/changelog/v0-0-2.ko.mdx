v0.0.2는 커뮤니티 리뷰, 확장된 언어 지원 및 AI 도구를 더 효과적으로 발견하고 공유할 수 있도록 돕는 다양한 개선 사항으로 v0.0.1을 기반으로 합니다.

### 하이라이트

- **도구 리뷰**: 1–5점으로 도구 평가 및 리뷰 작성
- **9개 언어**: 영어, 중국어, 한국어, 일본어, 포르투갈어, 스페인어, 독일어, 프랑스어, 베트남어 지원
- **도구 제출**: Free 및 Pro 플랜 옵션으로 AI 도구 제출
- **더 나은 브라우징**: 페이지네이션, 개선된 검색, 더 명확한 도구 페이지

### 새 기능

- **도구 리뷰**: 1–5점 평가, 선택적 텍스트 리뷰, 다른 사용자의 평가 및 댓글 보기
- **다국어 지원**: 9개 언어, 사이트 헤더에서 언어 전환
- **도구 제출**: Free/Pro 플랜 선택, 승인 팁이 있는 명확한 양식
- **브라우징 및 발견**: 페이지네이션된 도구 목록, URL 유지 검색

If you have suggestions for tools to add or features to improve, please reach out at support@buildway.cc.
