v0.0.1은 BuildWay의 첫 공개 버전입니다. 우리의 목표는 간단합니다. 산재된 리소스의 소음과 혼란 없이 모든 요구에 맞는 최고의 AI 도구를 발견할 수 있도록 돕는 것입니다.

### 하이라이트

- **선별된 AI 도구 디렉토리**: 10개 이상의 카테고리에서 59개 이상의 신중하게 선별된 도구
- **스마트 검색 및 발견**: 이름, 카테고리 또는 기능으로 도구 검색
- **다국어 경험**: 첫날부터 완전한 영어 및 중국어 지원
- **깔끔하고 현대적인 인터페이스**: 중요한 것에 집중—올바른 도구 찾기

### 새 기능

- **도구 디렉토리**: 카테고리별로 구성된 59개 이상의 AI 도구 탐색
- **검색 및 필터**: 이름, 설명 또는 태그로 도구 검색
- **도구 상세**: 전체 설명, 별점, 사용자 피드백, 공식 사이트 링크
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 최적화
- **다크 모드**: 모든 조명 조건에서 편안한 보기

If you have suggestions for tools to add or features to improve, please reach out to us at support@buildway.cc.
