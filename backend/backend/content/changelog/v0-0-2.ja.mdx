v0.0.2は、コミュニティレビュー、拡張された言語サポート、AIツールをより効果的に発見・共有できるよう支援する多くの改善で、v0.0.1を基盤としています。

### ハイライト

- **ツールレビュー**: 1–5星でツールを評価し、レビューを書く
- **9言語**: 英語、中国語、韓国語、日本語、ポルトガル語、スペイン語、ドイツ語、フランス語、ベトナム語をサポート
- **ツール提出**: FreeおよびProプランオプションでAIツールを提出
- **より良いブラウジング**: ページネーション、改善された検索、より明確なツールページ

### 新機能

- **ツールレビュー**: 1–5星評価、オプションのテキストレビュー、他ユーザーの評価とコメントを表示
- **多言語サポート**: 9言語、サイトヘッダーから言語切り替え
- **ツール提出**: Free/Proプラン選択、承認のヒント付き明確なフォーム
- **ブラウジングと発見**: ページネーション付きツールリスト、URL保持検索

If you have suggestions for tools to add or features to improve, please reach out at support@buildway.cc.
