v0.0.2 builds on v0.0.1 with community reviews, expanded language support, and a host of improvements to help you discover and share AI tools more effectively.

### Highlights

- **Tool reviews**: Rate tools 1–5 stars and write reviews—your feedback helps others choose
- **9 languages**: Full support for English, 中文, 繁體中文, 한국어, 日本語, Português, Español, Deutsch, Français, and Tiếng Việt
- **Tool submission**: Submit your AI tools with Free and Pro plan options
- **Better browsing**: Pagination, improved search, and clearer tool pages
- **Stronger discoverability**: Structured data for search engines so tools are easier to find

### New features

- **Tool Reviews**
  - Rate any tool with 1–5 stars
  - Add optional written reviews to share your experience
  - View ratings and comments from other users on each tool page
  - Draft auto-save so you don't lose your work
  - Login required to submit—keeps feedback trustworthy
- **Multi-language Support**
  - 9 languages: English, Chinese (Simplified & Traditional), Korean, Japanese, Portuguese, Spanish, German, French, and Vietnamese
  - Switch language from the site header
  - Tool content and UI adapt to your chosen language
- **Tool Submission**
  - Submit your AI tools for inclusion in the directory
  - Choose between Free and Pro plans with different benefits
  - Clear form with tips for faster approval
- **Browsing & Discovery**
  - Paginated tool list for easier navigation
  - Search with one-click clear and URL persistence
  - Improved layout for featured and popular tools
- **Tool Detail Pages**
  - Reviews section integrated into each tool page
  - Clearer content structure and heading hierarchy
  - Average rating and review count displayed

### Improvements

- **Search**: Clear button when typing, search query preserved in URL for sharing
- **Pagination**: Jump to specific pages, first/last shortcuts on long lists
- **SEO**: Schema.org structured data on tool pages for better search visibility
- **UI polish**: Loading states, responsive layout tweaks, and smoother interactions

### What's next

- More tools: expanding our directory to 100+ tools
- Enhanced search: improved algorithms and filters
- Tool comparisons: side-by-side comparisons of similar tools
- Collections: curated lists for specific use cases
- Advanced filters: price range, features, and more

If you have suggestions for tools to add or features to improve, please reach out at support@buildway.cc. Your feedback helps us build a better platform.
