### What is AIEditor
AIEditor is a next-gen rich text editor powered by AI. It is designed to help users build knowledge products quickly and efficiently.

### Features
- ### Out of the box: AIEditor requires minimal preparation and can be up and running with just a few lines of code.
- ### Markdown Friendly: It can recognize and render basic Markdown syntax accurately, providing real-time results.
- ### Powerful AI capabilities: AIEditor can check for spelling and grammar errors, expand or condense word counts, provide one-click translations, and summarize content seamlessly within the editor.
- ### Collaboration: Multiple users can work on the same document simultaneously, allowing real-time visibility into each other's input and changes.
- ### Commenting: Reviewers can add comments or suggestions to specific sections of the document, flag errors, or highlight areas for improvement.
- ### Integration: AIEditor integrates various open-source tools like virtual whiteboard, hand-drawn sketches, and chart editor for enhanced functionality.

### How AIEditor Works
AIEditor is developed based on Web Component and supports any framework. It offers server-side support for private deployment, ensuring data security and privacy. The editor supports access to LLMs and provides a private Apikey for enhanced data control.

### Price
- ### Open Source Version: AIEditor is available under the more liberal LGPL license with no user or application limits.
- ### Pro Version: For a low price, users can access the pro version with unlimited users and applications, released under the LGPL-v2.1 License.
