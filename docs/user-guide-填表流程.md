# 填表流程用户指南

## 准备工作

### 1. 添加网站资料

1. 打开扩展的设置页面（Options）
2. 在"网站管理"区域添加你的网站信息：
   - 网站名称（如：EaseUS）
   - 网站 URL（如：https://www.easeus.com）
   - 邮箱地址
   - 其他信息（作者名、简介等）
3. **重要：确保勾选"启用"选项**
4. 保存

### 2. 添加外链

1. 在设置页面的"外链管理"区域
2. 添加要提交的外链 URL
3. 可以添加关键词、备注等信息
4. 保存

## 填表流程

### 方式一：快速填充（推荐）

1. 打开要填表的页面
2. 打开 Side Panel（侧边栏）
3. 在"填表"标签页中：
   - 系统会自动检测当前页面是否有表单
   - 智能匹配会自动推荐合适的外链
   - 在"我的网站"区域选择要使用的网站
4. 点击"一键填充"按钮
5. 系统会自动填充表单并记录提交

### 方式二：手动选择

1. 打开 Side Panel
2. 切换到"外链"标签页
3. 搜索或浏览外链列表
4. 点击要使用的外链
5. 切换回"填表"标签页
6. 在"我的网站"区域选择网站
7. 点击"一键填充"

## 查看统计

1. 切换到"我的网站"标签页
2. 查看每个网站的统计信息：
   - 外链总数
   - 已提交数量
   - 已审核数量
   - 待提交数量

## 常见问题

### Q: 为什么看不到我的网站？

A: 请检查：
1. 是否已经添加网站资料
2. 网站是否已启用（enabled = true）
3. 尝试刷新 Side Panel

### Q: 如何知道哪个网站被选中了？

A: 在"填表"标签页的"我的网站"区域，下拉框会显示当前选中的网站，下方会显示该网站的详细信息。

### Q: 填表后如何确认已记录？

A:
1. 填表成功后会显示成功提示
2. 在"我的网站"标签页可以看到统计数字更新
3. 系统会自动记录提交时间和状态

### Q: 智能匹配是如何工作的？

A: 系统会根据以下因素自动匹配外链：
- 当前页面的 URL 和域名
- 页面内容和关键词
- 外链的关键词和备注
- 匹配置信度会显示在界面上
