### 触发条件

- 严重错误导致功能不可用
- 性能严重下降
- 用户大量投诉

### 回滚步骤

1. **立即回滚**
   ```bash
   # 回滚到上一个版本
   git revert <commit-hash>
   npm run build
   ```

2. **通知用户**
   - 发布公告说明情况

   - 分析问题原因

4. **重新发布**
